# Anak Agung Gede Mahardika Juliananda | Digital Marketing Professional

Premium personal portfolio website showcasing expertise in Digital Marketing, Business Development, and Strategic Operations.

**Live:** [mahardikaagng.github.io](https://mahardikaagng.github.io)

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React Icons
- Next.js Image Optimization

## Features

- 🎨 **Premium Dark Luxury Design** with gold and electric blue accents
- ✨ **Smooth Animations** with Framer Motion for tasteful motion effects
- 📱 **Fully Responsive** across desktop, tablet, and mobile devices
- 🎯 **Complete SEO** with metadata, Open Graph, and Schema.org structured data
- ♿ **Accessible Components** with proper semantic HTML and ARIA labels
- 🚀 **Performance Optimized** with Next.js optimizations and image handling
- 📊 **Comprehensive Sections**: Hero, About, Experience, Projects, Skills, Achievements, Contact

## Sections

- **Hero**: Introduction with animated profile image, professional headline, and CTAs
- **About**: Professional summary, location, education, and certifications
- **Experience**: Animated timeline of work experience and projects
- **Projects**: Portfolio showcase with projects including Santai Seaplane, Tetuek, Slimlicious, and Moto Rent
- **Achievements**: Key metrics and measurable impact statistics
- **Skills**: Categorized professional competencies across 6 domains
- **Contact**: Complete contact information with social links

## Development

```bash
npm install
npm run dev      # Start development server
npm run build    # Production build
npm start        # Start production server
npm run lint     # Run ESLint
```

Visit `http://localhost:3000` to see the site.

## Deployment

Deployed on GitHub Pages via Next.js static export.
